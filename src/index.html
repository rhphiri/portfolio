Robofriends

This interesting project taught me a lot about how to use React and Bootstrap 5 in designing a website.

I also learned a great deal about how to get a React app to interact with various APIs in order to accomplish a certain goal.

The website Robofriends tries to answer one odd but interesting question: What would our friends look like if they were robots?

What the website does is it fetches user data i.e. friend details, from one API and then communicates with another API in manipulating that data. The manipulated data is then displayed on the website as robots.

Object Recognition App

In this project, I learned how to incorporate AI into a website. I also learned how to do the frontend, backend and create a database.

To create the frontend, I used React, Bootstrap 5, HTML5 and CSS3. To create the backend, I used Nodejs and Express. As for the database, I used PostgreSQL.

This website takes an input in the form of a url to an image submitted by the user. The website then communicates with an AI called Clarifai through a backend server about the image. Any major objects found in the image are immediately pointed out to the user.
